# 人工智能基础知识

## 什么是机器学习
机器学习是人工智能的一个子领域，让计算机通过数据学习，而不需要明确编程。
常见应用：推荐系统、图像识别、垃圾邮件过滤。

## 什么是深度学习
深度学习是机器学习的子集，使用多层神经网络。
代表技术：CNN（卷积神经网络）用于图像处理，RNN用于序列数据。

## 什么是大语言模型
大语言模型（LLM）是深度学习的应用，通过海量文本训练，能够理解和生成人类语言。
例子：GPT、DeepSeek、Claude。

## 什么是 RAG
RAG（检索增强生成）是一种技术，让大模型在生成回答前，先从知识库中检索相关信息。
优点：回答更准确、可更新知识、减少幻觉。

## 什么是 Agent
Agent 是一个能感知环境、做出决策、使用工具并采取行动的智能实体。
Agent = 大模型 + 工具 + 记忆 + 规划。
